# Chapter 7: 機械学習による文書分類

## 概要
SST-2（Stanford Sentiment Treebank）データセットを使った感情分析の機械学習問題です。

## 問題一覧

| 問題番号 | タイトル | 内容 |
|----------|----------|------|
| 60 | データの読み込み | 学習・検証データのラベル分布を確認 |
| 61 | Bag of Words特徴量 | テキストをBoW（単語頻度辞書）形式に変換 |
| 62 | 学習 | ロジスティック回帰モデルを学習 |
| 63 | 予測 | 検証データの先頭事例を予測 |
| 64 | 条件付き確率 | 各ラベルの条件付き確率 P(Y|X) を算出 |
| 65 | 任意テキストの予測 | 自由入力テキストの感情を予測 |
| 66 | 混同行列 | Confusion Matrixを作成（TN/FP/FN/TP）|
| 67 | 評価指標 | 正解率・適合率・再現率・F1スコアを計算 |
| 68 | 特徴量の重み | ポジティブ/ネガティブ寄りの単語Top20を抽出 |
| 69 | 正則化パラメータ | Cを変えながら精度の変化をグラフ化 |

## 学習ポイント
- scikit-learnによる機械学習パイプライン
- `DictVectorizer`による特徴量変換
- ロジスティック回帰（`LogisticRegression`）
- 評価指標（Accuracy, Precision, Recall, F1）
- 混同行列の理解
- 正則化パラメータのチューニング
- joblibによるモデルの永続化
