# Chapter 1: 準備運動

## 概要
Pythonの基本的な文字列操作を学ぶための準備運動問題です。

## 問題一覧

| 問題番号 | タイトル | 内容 |
|----------|----------|------|
| 00 | 文字列の逆順 | 「stressed」を逆順にして「desserts」を得る |
| 01 | 「パタトクカシーー」 | 文字列から奇数番目の文字を抽出 |
| 02 | 「パトカー」＋「タクシー」＝「パタトクカシーー」 | 2つの文字列を交互に連結 |
| 03 | 円周率 | 文章中の単語の文字数をリストで取得 |
| 04 | 元素記号 | 元素周期表の文章から元素記号と位置の辞書を作成 |
| 05 | n-gram | 文字n-gramと単語n-gramを生成する関数の実装 |
| 06 | 集合 | bi-gramを使った集合演算（和・積・差）|
| 07 | テンプレートによる文生成 | 引数を埋め込んだ文字列を生成する関数 |
| 08 | 暗号文 | 小文字アルファベットを(219-文字コード)で変換する暗号化 |
| 09 | Typoglycemia | 単語の先頭と末尾以外の文字をランダムにシャッフル |

## 学習ポイント
- 文字列のスライス・インデックス操作
- リスト内包表記
- 辞書（dict）の操作
- 集合（set）演算
- f文字列によるフォーマット
- `random` モジュールの使用
