# Chapter 2: UNIXコマンド

## 概要
テキストファイル（popular-names.txt）を題材に、UNIXコマンドと同等の処理をPythonで実装する問題です。

## 問題一覧

| 問題番号 | タイトル | 内容 | 対応UNIXコマンド |
|----------|----------|------|------------------|
| 10 | 行数のカウント | ファイルの行数を数える | `wc -l` |
| 11 | タブをスペースに置換 | タブ文字をスペースに変換 | `tr '\t' ' '` / `expand` |
| 12 | col1.txt と col2.txt | 1列目と2列目を別ファイルに抽出 | `cut -f 1,2` |
| 13 | col1.txtとcol2.txtをマージ | 2つのファイルを横に連結 | `paste` |
| 14 | 先頭からN行を出力 | 先頭N行を表示 | `head -n N` |
| 15 | 末尾のN行を出力 | 末尾N行を表示（dequeを使用） | `tail -n N` |
| 16 | ファイルをN分割 | ファイルを指定数に分割 | `split -n` |
| 17 | １列目の文字列の異なり | 1列目のユニークな値を抽出 | `cut \| sort \| uniq` |
| 18 | 各行を3コラム目の数値の降順にソート | 3列目の値でソート | `sort -k3,3nr` |
| 19 | 各行の1コラム目の文字列の出現頻度 | 名前の出現回数をカウント | `cut \| sort \| uniq -c` |

## 学習ポイント
- ファイルの読み書き操作
- `collections.deque` によるメモリ効率の良い処理
- `collections.Counter` による頻度カウント
- ソート処理（`sorted()`, `key`引数）
- ラムダ式の活用
