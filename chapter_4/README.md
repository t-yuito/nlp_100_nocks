# Chapter 4: 形態素解析

## 概要
spaCy（GiNZA）を使った日本語形態素解析の問題です。「走れメロス」の冒頭部分やWikipediaコーパスを対象に解析を行います。

## 問題一覧

| 問題番号 | タイトル | 内容 |
|----------|----------|------|
| 30 | 形態素解析結果の読み込み | spaCy/GiNZAで動詞を抽出 |
| 31 | 動詞の表層形と基本形 | 動詞の活用形と原形（lemma）を抽出 |
| 32 | 名詞句の抽出 | 「AのB」形式の名詞句を抽出 |
| 33 | 係り受け解析 | 各単語とその係り先（head）を出力 |
| 34 | 「メロス」の述語 | 主語が「メロス」の文から述語を抽出 |
| 35 | 係り受け木の可視化 | displacyで係り受け木をSVG出力 |
| 36 | 単語の出現頻度 | Wikipediaコーパスの単語頻度Top20 |
| 37 | 頻度上位の名詞 | 名詞・固有名詞の頻度Top20 |
| 38 | TF-IDF | 「日本」記事のTF-IDF上位語を抽出 |
| 39 | Zipfの法則 | 単語の出現頻度と順位の関係を可視化 |

## 学習ポイント
- spaCy/GiNZAによる形態素解析
- 品詞（POS）タグの利用
- 係り受け解析（dependency parsing）
- TF-IDFの計算
- `Counter`による頻度集計
- matplotlibによる可視化
- ジェネレータ（`yield`）の活用
