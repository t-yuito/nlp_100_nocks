# Chapter 5: 大規模言語モデル (LLM)

## 概要
Google Gemini APIを使った大規模言語モデルの活用問題です。プロンプトエンジニアリングやLLMの特性を学びます。

## 問題一覧

| 問題番号 | タイトル | 内容 |
|----------|----------|------|
| 40 | Zero-shot推論 | 例示なしで歴史の並び替え問題を解かせる |
| 41 | Few-shot推論 | 複数の例題を提示して精度を向上 |
| 42 | ベンチマーク評価 | JMMLUデータセットで多肢選択問題の正解率を計測 |
| 43 | 選択肢のバイアス | 正解を常にDに固定して位置バイアスを検証 |
| 44 | 推論タスク | 東急線の乗り換え問題を解かせる |
| 45 | マルチターン対話 | チャット履歴を保持した連続対話 |
| 46 | 川柳生成 | お題に沿った川柳を生成 |
| 47 | 生成物の評価 | 生成した川柳をLLM自身に評価させる |
| 48 | プロンプトインジェクション | 不正なプロンプト注入の影響を検証 |
| 49 | トークンカウント | テキストのトークン数を計測 |

## 学習ポイント
- Google Generative AI (Gemini) APIの使用
- プロンプトエンジニアリング（Zero-shot / Few-shot）
- マルチターン対話の実装
- ベンチマークデータセット（JMMLU）の活用
- プロンプトインジェクション攻撃の理解
- トークナイゼーションの概念
