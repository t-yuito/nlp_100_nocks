# Chapter 3: 正規表現

## 概要
WikipediaのJSONデータ（jawiki-country.json）から「イギリス」の記事を抽出し、正規表現を使ってMediaWikiマークアップを解析する問題です。

## 問題一覧

| 問題番号 | タイトル | 内容 |
|----------|----------|------|
| 20 | JSONデータの読み込み | pandasでJSONを読み込み、イギリスの記事を抽出 |
| 21 | カテゴリ名を含む行を抽出 | `[[Category:...]]` を含む行を抽出 |
| 22 | カテゴリ名の抽出 | カテゴリ名のみを取り出す |
| 23 | セクション構造 | 見出し（=で囲まれた部分）とそのレベルを抽出 |
| 24 | ファイル参照の抽出 | 記事中で参照されているメディアファイル名を抽出 |
| 25 | テンプレートの抽出 | 基礎情報テンプレートのフィールド名と値を辞書化 |
| 26 | 強調マークアップの除去 | `''`や`'''`による強調を除去 |
| 27 | 内部リンクの除去 | `[[リンク|表示文字]]`形式のリンクを除去 |
| 28 | MediaWikiマークアップの除去 | 複数のマークアップを一括除去 |
| 29 | 国旗画像のURLを取得 | Wikimedia APIを使って国旗画像のURLを取得 |

## 学習ポイント
- pandasによるJSON読み込み（`pd.read_json()`）
- 正規表現の基本（`re.search()`, `re.findall()`, `re.sub()`）
- 非貪欲マッチ（`?`）
- グループ化と後方参照
- `requests`ライブラリによるAPI呼び出し
